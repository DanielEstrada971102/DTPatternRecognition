The dtpr.base module provides a collection of classes designed to simplify and enhance the manipulation of Event data extracted from ROOT TTrees of NTuples. These classes enable efficient extraction, transformation, and analysis of event information.

Important

The base classes handle reading and processing data from input files using a central configuration file, such as run_config.yaml. Throughout the documentation, you will find details on the configurable parameters that allow support for various data formats.
